COMO CRIAR OS ICONES DA EXTENSAO
=================================

A extensao precisa de icones em 3 tamanhos:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

OPCAO 1 - Usar ferramenta online:
1. Abra https://svgtopng.com/
2. Faça upload do arquivo icon.svg que está em app/frontend/public/
3. Converta para os 3 tamanhos
4. Salve os arquivos nesta pasta (extension/icons/)

OPCAO 2 - Criar seu próprio icone:
1. Use qualquer editor (Canva, Figma, Paint, etc)
2. Crie um icone quadrado verde com um alvo/target
3. Salve nos 3 tamanhos mencionados

Por enquanto, a extensao funcionara sem os icones, mas terá um icone generico.
